- Git versions prompts, not the runtime (model version, tool schemas, retriever state, routing logic).
- Traces explain failures, but don‚Äôt reliably let you reproduce a working state.

When snapshots make sense
If you use RAG, tools, LangGraph state, or deploy often, you need more than prompt diffs. ‚ÄúWorked yesterday, broke today‚Äù usually means something outside the prompt changed.

Minimum useful ‚Äúagent snapshot‚Äù
