1. **Yes**, you basically go it !  (you have to restart the app after modification)
   1. Edit your config file
   2. Restart the compose
2. Your custom Paths "C:/Test" and "C:/Family Photos" are **mounted to the same destination** directory  ("/photoprism/originals"), which is already used by the default "Pictures" folder.

What you need to do :  
Change the destination for your additional folders, for example :  
 `- "C:Test:/photoprism/originals/test"`   
 `- "C:Family Photos:/photoprism/originals/family"`   
 `- "C:RandomPhotosDirectory:/photoprism/originals/whatevernameiwant"`   
*(you can name the destination folder as you want, the directory will be created if it doesn't exist)*
