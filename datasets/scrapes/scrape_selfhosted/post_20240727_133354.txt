Comment 3:
  Author: deelayman
  Comment: For transcoding, QSV is probably the better option. Especially if you'll be hosting a game server which I assume to mean streaming gameplay to remote devices. I'm recalling something about a low limit on active NVENC streams on most NVIDIA GPU's (like 5 or something) - and this allotment is likely to be shared between encodes for media and active game sessions. Additionally, if you want to host any local AI models a GPU seems to be a semi-requirement.

So a GPU is a good idea for some things you might want to do. But transcoding may be better served by an Intel CPU with QSV
